# 🎬 Movie Streaming Web App (Netflix-Inspired)

🔗 **Live Demo:** [https://movie-netflix-app.vercel.app/](https://movie-netflix-app.vercel.app/)

---

## 🚀 Overview

A modern, high-performance movie streaming web application inspired by Netflix, built using Next.js.
The app delivers a smooth browsing experience with dynamic content, responsive UI, and scalable architecture.

---

## ✨ Key Features

* 🎥 Browse trending, popular, and categorized movies
* 🔍 Search functionality for discovering content
* 📱 Fully responsive design (mobile-first)
* ⚡ Fast rendering using SSR/CSR hybrid approach
* 🎨 Clean, modern UI inspired by Netflix
* 🔤 Optimized typography using `next/font`
* 🌐 API integration for real-time movie data

---

## 🧠 Tech Stack

* **Frontend:** Next.js, React, JavaScript
* **Styling:** CSS / Tailwind (if used, you can confirm)
* **API:** TMDB (The Movie Database) *(assumed — confirm if needed)*
* **Deployment:** Vercel

---

## 🏗️ Architecture Highlights

* App Router (`app/` directory structure)
* Component-based scalable UI
* API abstraction layer for clean data handling
* Optimized font loading via `next/font`
* Separation of UI and data logic for maintainability

## ⚙️ Getting Started

Clone the repo:

```bash
git clone https://github.com/AmanBang/movieWebsite.git
cd movieWebsite
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open 👉 [http://localhost:3000](http://localhost:3000)
